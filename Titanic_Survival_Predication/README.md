# 🚢 Titanic Survival Prediction

## 📌 About The Project
This project builds a Machine Learning model to predict whether a passenger survived the Titanic disaster. It is a classic beginner-friendly classification problem widely used for learning ML concepts.

---

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-Learn

---

## 📂 Dataset
The dataset includes passenger information such as:

- Age
- Gender
- Passenger Class
- Fare
- Embarkation Port
- Family Information
- Survival Status

---

## ⚙️ Project Workflow

1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Encoding Categorical Variables
5. Train-Test Split
6. Model Training (Logistic Regression)
7. Model Evaluation

---

## 📊 Model Performance

- Algorithm: Logistic Regression
- Accuracy: ~75% – 85%
- Evaluation Metrics:
  - Confusion Matrix
  - Classification Report

---

## 🔍 Key Insights

- Females had a significantly higher survival rate.
- First-class passengers were more likely to survive.
- Fare amount positively correlated with survival.
